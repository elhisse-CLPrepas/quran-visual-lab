Quran Visual Lab

Générateur intelligent de visuels coraniques et d'outils d'apprentissage

La paix durable ne se décrète pas. Elle se construit d'abord dans la conscience.

Vision du projet

Créer une application web qui permet :

  • Accéder rapidement à un verset du Coran
  • Générer automatiquement un prompt visuel professionnel
  • Produire des illustrations infographiques spirituelles
  • Aider à l’apprentissage et la mémorisation du Coran
  • Faciliter la publication pédagogique et spirituelle

L'utilisateur saisit simplement : sourate + verset. L'application se charge du reste.

✨

Flux simplifié

Utilisateur → saisie S+V → récupération verset → génération Prompt Maître → génération visuel → export.

Objectif : supprimer le copier-coller et standardiser la création de visuels.

Objectif 1

Supprimer le copier-coller du texte coranique.

Objectif 2

Créer automatiquement un prompt maître standardisé.

Objectif 3

Générer des visuels pédagogiques et spirituels.

Modules & génération automatique

🖼️ Module visuel & prompt

Une fois le verset récupéré, l'application génère :

  • Prompt visuel pour générateur d'image.
  • Prompt pédagogique pour fiche d'apprentissage.
  • Prompt publication pour blog ou WhatsApp.

Carte visuelle spirituelle : Titre, nom sourate, bloc verset, décor motifs islamiques, lumière, mosquée, calligraphie.

Palette :

📚 Module apprentissage & méditation

Lecture – affichage clair · Écoute – audio du verset · Répétition 3 fois · Mémorisation (masquer mots, quiz).

Méditation : thème du verset, leçon spirituelle, question de réflexion.

Export : copier verset, télécharger prompt/fiche/image, partager WhatsApp/FB/Insta.

📐 Formats visuels disponibles
1080×1920 1080×1080 A4 pédagogique

⚙️ Architecture technique

Front-end : HTML, CSS, JavaScript, Bootstrap.

Back-end : Option simple (API Coran) ou avancée (Node.js/PHP avec cache).

Source des données : API coranique (ex. AlQuran.cloud) pour récupérer nom sourate, texte arabe, traduction.

Vérification : validation sourate/verset + message d'erreur clair.
